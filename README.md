# Aastha Bajpai — Portfolio Website

A modern, responsive personal portfolio for **Aastha Bajpai** — MCA student at NIT Jamshedpur and Full-Stack (MERN) Developer.

🔗 **Live Site**: [aasthabajpai010.github.io](https://aasthabajpai010.github.io)

---

## Features

- **Professional Design** — Space Grotesk typography, electric teal–purple palette, dot-grid hero background
- **Dark / Light Mode** — Toggle in navbar, preference saved via `localStorage`, auto-detects system theme
- **Fully Responsive** — Mobile, tablet, and desktop layouts
- **Skill Pills** — Technologies grouped by category (Languages / Frontend / Backend / Databases / CS Fundamentals / Tools)
- **Real Projects** — AI-Integrated College ERP, BhashaShikho, House Price Prediction — each with GitHub links
- **Achievements** — NIMCET AIR 519, Postman certification, HackerRank / LeetCode / GeeksforGeeks profiles
- **Resume Download** — Direct PDF download from hero section

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Grid, Flexbox, CSS custom properties, animations
- **JavaScript** — Vanilla JS, no frameworks
- **Google Fonts** — Space Grotesk + Inter
